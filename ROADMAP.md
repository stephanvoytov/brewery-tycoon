# Brewery Tycoon — Roadmap

## Что уже хорошо
- Модель данных: GameState → Brewery → Recipes/Batches/Staff
- Пивоваренный процесс: 4 стадии
- 12 стилей с разными параметрами
- Сезонность
- SVG-визуализация цеха
- Выбор конкретных ингредиентов в рецепте
- 5 валют

---

## Sprint 1: Фундамент — цель и баланс

- [ ] **1. Система целей / достижений**
  - Цель-минимум: «Заработайте $100,000» (счётчик на дашборде)
  - Milestone-достижения (первая партия, первый сотрудник, первый контракт)
  - Каждое достижение даёт бонус (репутация +5, скидка на апгрейд)

- [ ] **2. Game Over**
  - Если `money < -$2,000` → игра окончена
  - Опция начать заново с половинным капиталом

- [ ] **3. Ребаланс экономики**
  - Стартовый капитал: $10,000 → **$3,000**
  - Аренда: $50/день → **$100/день**
  - Стартовые ингредиенты: 50 кг → **20 кг** каждого
  - Кредит: 0.5%/день → **1%/день**, автосписание при превышении лимита

- [ ] **4. Порча ингредиентов**
  - Каждый день -0.5% от всех ингредиентов (списание)
  - Вынуждает покупать с умом

- [ ] **5. Уведомления**
  - Партия готова → всплывающее сообщение
  - Исследование завершено → уведомление
  - Просрочка контракта → предупреждение за 3 дня

---

## Sprint 2: Контент и глубина

- [ ] **6. Система случайных событий (10%/день)**
  - «Жара!» — спрос на пшеничное +50% на 5 дней
  - Поломка котла — ремонт $500 или простой 2 дня
  - Хмель дорожает +20% на неделю
  - Пивной фестиваль — репутация +10
  - Налоговая проверка — штраф $500

- [ ] **7. Профиль вкуса пива**
  - Идеальный профиль для каждого стиля (IBU, ABV, SRM, тип солода/хмеля/дрожжей)
  - Процент соответствия в редакторе рецепта
  - Качество = близость к идеалу

- [ ] **8. Конкуренты**
  - 3-5 AI-пивоварен с долей рынка
  - Продают случайный объём каждый день
  - Игрок отъедает долю при высоком качестве/репутации
  - График доли рынка

- [ ] **9. Контракты с фильтрами**
  - Бары: мало, дорого
  - Магазины: много, дёшево
  - Рестораны: среднее, требования к качеству

---

## Sprint 3: Полировка

- [ ] **10. FAQ / туториал**
  - Первый вход: 5 шагов (ингредиенты → рецепт → варка → контракт → продажа)
  - Подсказки на каждой странице

- [ ] **11. История цен**
  - График цены продажи по дням + средняя себестоимость

- [ ] **12. Прогресс пивоварни (level)**
  - Уровень растёт от суммарного дохода
  - Открывает новое оборудование и больше контрактов

- [ ] **13. Визуальные/звуковые мелочи**
  - Звуки: готово пиво, новый день, деньги
  - Улучшенные уведомления

---

## Балансовые ориентиры
- Игрок должен **чувствовать нехватку денег** первые 30-50 дней
- Одна ошибка (неудачная партия, просрочка контракта) должна быть **болезненной, но не фатальной**
- К 100-му дню — стабильный рост, к 200-му — win condition
- Наполнение: активное решение каждые 2-3 дня, а не только «Новый день»
