# Пивоваренный Тайкун (Brewery Tycoon)

Экономическая стратегия — управляйте собственной пивоварней. Варите пиво, заключайте контракты, нанимайте персонал, улучшайте оборудование и исследуйте новые технологии.

---

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Бэкенд    | Python 3.10+, FastAPI, SQLAlchemy 2.0, Pydantic 2 |
| База данных | SQLite |
| Фронтенд  | Vanilla JavaScript (SPA), CSS (тёмная тема) |
| Сервер    | Uvicorn |

---

## Быстрый старт

```bash
# Установить зависимости
pip install -r backend/requirements.txt

# Запустить сервер
uvicorn backend.main:app --reload --port 8000
```

Откройте **http://localhost:8000** в браузере.

При первом запуске база данных `backend/brewery.db` создаётся автоматически, и игра стартует с начальным балансом \$10 000.

## API (основные эндпоинты)

Все эндпоинты принимают `game_id` как query-параметр.

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/game/new` | Создать новую игру |
| GET | `/api/game/state?game_id=` | Полное состояние игры |
| POST | `/api/game/tick?game_id=&days=` | Прожить N дней |
| GET | `/api/brewery/?game_id=` | Данные пивоварни |
| POST | `/api/brewery/upgrade` | Улучшить пивоварню |
| GET | `/api/recipes/?game_id=` | Список рецептов |
| POST | `/api/recipes/{id}/brew` | Начать варку |
| GET | `/api/batches/?game_id=` | Список партий |
| POST | `/api/batches/{id}/sell` | Продать партию |
| POST | `/api/inventory/buy` | Купить ингредиенты |
| POST | `/api/inventory/equipment/buy` | Купить оборудование |
| POST | `/api/staff/hire` | Нанять сотрудника |
| POST | `/api/staff/{id}/fire` | Уволить сотрудника |
| POST | `/api/staff/{id}/train` | Обучить сотрудника |
| GET | `/api/market/?game_id=` | Условия рынка |
| GET | `/api/market/contracts?game_id=` | Контракты |
| POST | `/api/market/contracts/{id}/sign` | Подписать контракт |
| GET | `/api/research/?game_id=` | Дерево исследований |
| POST | `/api/research/{id}/start` | Начать исследование |

Полная документация — `http://localhost:8000/docs` (Swagger UI).

## Геймплей

### 🏭 Пивоварня
- Улучшайте количество варочных котлов, ферментеров, хранилище
- Откройте тапрум для розничных продаж
- Покупайте оборудование: котлы, линии розлива, системы охлаждения

### 📝 Рецепты и варка
- 12 встроенных стилей пива (Лагер, Эль, Стаут, IPA, ...)
- Создавайте собственные рецепты
- Контролируйте ABV, IBU, себестоимость

### 🛢 Партии
- Партия проходит 5 стадий: затирание → кипячение → ферментация → дозревание → готова
- Качество пива зависит от сложности рецепта, оборудования и случайных факторов

### 💰 Рынок и контракты
- Изучайте рыночный спрос и сезонные колебания
- Заключайте контракты с барами, ресторанами и магазинами
- Штрафы за срыв сроков

### 👥 Персонал
- Нанимайте пивоваров, продавцов и администраторов
- Повышайте навыки сотрудников обучением
- Следите за моралью

### 🔬 Исследования
- Открывайте новые технологии: автоматизация, контроль качества, маркетинг
- Исследования требуют времени и денег

### 📈 Финансы
- Отслеживайте доходы, расходы, чистую прибыль
- Графики за последние 30 дней
- Кредиты при отрицательном балансе

```bash

pip install -r backend/requirements.txt
```
