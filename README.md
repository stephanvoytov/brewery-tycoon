# Brewery Tycoon — Пивоваренный Тайкун 🍺

Экономическая стратегия в реальном времени. Управляйте крафтовой пивоварней: варите пиво, заключайте контракты, нанимайте персонал, улучшайте оборудование. Достигните $100,000 выручки и станьте пивным магнатом.

**Live:** [brewery-tycoon.mooo.com](https://brewery-tycoon.mooo.com)

---

## 🎯 Геймплей

### Цель игры
Заработайте **$100,000** общей выручки. Прогресс-бар на дашборде показывает приближение к победе. По пути открываются достижения, дающие постоянные бонусы.

### 🍺 Пивоварение
- **4 стадии:** затирание → кипячение → ферментация → дозревание
- Каждая партия проходит полный цикл; скорость зависит от оборудования и навыка пивоваров
- **12 стилей:** Лагер, Эль, Стаут, IPA, Портер, Пшеничное, Пильзнер, Кислый Эль, Бок, Пэйл Эль, Янтарный Эль, Бельгийский Трипель
- Выбор конкретных ингредиентов (солод, хмель, дрожжи) влияет на себестоимость и качество
- Качество пива стартует с 50; нерекомендованные ингредиенты снижают его

### 📋 Контракты и продажи
- Бары, рестораны и магазины предлагают контракты на поставку
- Штраф за срыв — 20% от суммы контракта
- **Предупреждение за 3 дня** до истечения срока
- Выполненные контракты увеличивают репутацию

### 👥 Персонал
- **Пивовар:** +3% скорости варки за единицу навыка
- **Продавец:** +2% к цене контрактов за единицу навыка
- **Администратор:** −2% к расходам за единицу навыка
- Найм, увольнение, обучение (skill × $200)

### 🏭 Улучшения и оборудование
- Варочные котлы, ферментеры, хранилище, тапрум, маркетинг
- 9 видов оборудования: котлы 50/100л, ферментеры, линии розлива/кегов, заторный чан, система охлаждения, лагерный танк
- Каждое улучшение повышает эффективность

### 🔬 Исследования
- 6 исследований: автоматизация варки, контроль качества, маркетинг в соцсетях, экскурсии, крафтовая линия, солодовня
- Требуют времени (2–7 дней) и денег ($2,000–$8,000)

### ⚠️ Риск банкротства
- При отрицательном балансе выдаётся кредит под **1%/день**
- Если долг превышает **$5,000** более **30 дней** — Game Over
- Можно начать заново с половиной оставшегося капитала или начать новую игру

### 🌾 Порча ингредиентов
- Ежедневно −0.5% всех запасов списывается
- Покупайте ровно столько, сколько нужно на ближайшие варки

## 🚀 Быстрый старт (локально)

```bash
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload --port 8000
```

Откройте **http://localhost:8000**. БД создаётся автоматически в `data/brewery.db`.

---

## 🐳 Деплой (Docker)

```bash
docker compose up -d --build
```

Сервис на `http://localhost:8000`. Для продакшена требуется reverse proxy с HTTPS (Caddy).

### Через GitHub

```bash
# На сервере:
```

## 📡 API

Полная документация — **http://localhost:8000/docs** (Swagger UI).

### Игра
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/game/new` | Создать игру |
| GET | `/api/game/state` | Полное состояние |
| POST | `/api/game/tick` | Прожить N дней |
| POST | `/api/game/restart-after-game-over` | Рестарт после банкротства |
| GET | `/api/game/saves` | Список сохранений |
| PUT | `/api/game/select` | Выбрать сохранение |
| POST | `/api/game/currency` | Сменить валюту |
